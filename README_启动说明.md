## 🚀 快速启动

### 方法一：桌面快捷方式（推荐）

我已经为您创建了桌面快捷方式，双击即可启动：

1. **🚀 启动脚本文案助手.command** - 启动完整服务器
2. **🔄 后台启动脚本文案助手.command** - 后台运行服务器
3. **🛑 停止脚本文案助手.command** - 停止服务器
4. **📊 检查脚本文案助手状态.command** - 查看运行状态

### 方法二：命令行启动

在项目目录中运行以下命令：

```bash
# 启动服务器（前台运行）
./start.sh

# 后台启动服务器
./start_background.sh

# 停止服务器
./stop.sh

# 查看状态
./status.sh
```

### 方法三：使用npm命令

```bash
# 安装依赖
npm install
cd client && npm install && cd ..

# 启动开发服务器
npm run dev
```

## 🛠️ 故障排除

### 问题1：端口被占用
```bash
# 检查端口占用
lsof -ti:5119
lsof -ti:3000

# 终止占用进程
kill -9 $(lsof -ti:5119)
kill -9 $(lsof -ti:3000)
```

### 问题2：依赖安装失败
```bash
# 清理缓存
npm cache clean --force
rm -rf node_modules
rm -rf client/node_modules

# 重新安装
npm install
cd client && npm install && cd ..
```

### 问题3：服务器无法启动
```bash
# 检查Node.js版本
node --version  # 需要 >= 18.0.0

# 检查npm版本
npm --version

# 查看详细错误
./status.sh
```
